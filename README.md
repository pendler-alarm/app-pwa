# Pendler-Alarm
This app is a PWA to show you the next routes according to your calendar.

## inspired by 
* Calendr: https://github.com/cbrasser/calendr
* @AbdullahiFatola: Draft of a searching app
* @jreinholz-code: idea of a pendler app


## Build Setup

``` bash
# 1. install dependencies
npm install

# 2. add client id from Google Cloud Console in your .env file
VITE_GOOGLE_CLIENT_ID=your_client_id_here

# 3. serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report
```

## used ressources
* OpenAI/ChatGPT/CODEX
* Google Calendar API
